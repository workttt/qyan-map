go语言入门

基础语法​

标准库文档 https://studygolang.com/pkgdoc

声明变量 var b bool = true var a string = "字符串" 声明基本变量可以省略变量名前后的var和类型 b := true a := "字符串"

基本变量：布尔型 数字类型(int int32 float32) 字符串类型 （Go的字符串是由单个字节连接起来的） （四种都是值类型-在栈中 特殊类型：派生类型 （数组，切片，map，指针，函数……） var 数组名 [大小数字] 类型 （固定长度的数据结构，声明时需要指定长度。）
var balance [10] float32 var balance = [5]float32{1000.0, 2.0, 3.4, 7.0, 50.0} var a [2][2] int a = [2][2] int{ {1, 2}, {3, 4} } 注意数组作为参数传递时的写法： 形参未设定数组大小： func getAverage(arr []int)

                        - 传递数组给函数时是按值传递，会复制整个数组。

切片，未指定大小的数组或者make创建 （动态长度的数据结构，基于数组构建。） var 切片名称 []类型 = make([]类型, len, capacity) var slice1 []type = make([]type, len) 简写 slice1 := make([]type, len) s :=[] int {1,2,3} 切片是对底层数组的一个引用，对切片的修改会影响原数组。

                        - 传递切片给函数时是按引用传递，不会复制整个切片。

总结：数组是固定长度的数据结构，而切片是动态长度的、对数组的引用。在实际开发中，切片更常用，因为它更灵活且方便管理。

map类型(一些语言的类似对象) var test = map[key的类型]val的类型 var test = map[int]int{} var test = map[int]int{1:11, 2:22} var test = map[string]int{"a": 11, "b": 22} // 创建一个具有初始值的map，其中键是string类型，值是int类型。 make(map[string]interface{}) 是在Go中创建一个空的map，其中键是string类型，值是interface{}类型，可以存储任意类型的值。

同时声明多个变量 var c, d int = 1, 2 var e, f = 123, "hello"

常量使用const声明，同var iota 0开始的特殊值

---------------------------- 字符串操作 byte是uint8的别名，通常用于处理ASCII字符。 rune是int32的别名，用于处理一个UTF-8字符或Unicode码点。

byteValue := byte('A')
fmt.Println(byteValue)  // 输出: 65

runeValue := rune('你')
fmt.Printf("%#U \n", runeValue)  // 输出: U+4F60
