gin入门

1.安装和使用​

安装 go get -u github.com/gin-gonic/gin

引入的包 import "github.com/gin-gonic/gin" import "net/http"

2.启动服务的端口配置​

r := gin.Default()                      // 1.创建路由
r.GET("/", func(c *gin.Context) {...})  // 2.绑定路由规则
r.Run(":8000")                          // 3.监听端口，默认在8080

3.请求方式​

r.GET("/ping", func)
r.POST("/ping", func)
r.PUT("/ping", func)
r.DELETE("/ping")
r.PATCH("/somePatch")

4.获取传参​

http:// ip : 端口 虚拟目录(/a/b/c...) 文件名部分() 锚部分() 参数部分()

http://localhost:8000/user/你的名字/asdasd?name=&queryParam=dddd&arrayParam=22&arrayParam=33

路由参数获取​

请求路径中 "/user/:name/:paramName" 虚拟目录部分参数获取

  • c.Param("name")
  • c.Param("paramName")

列表Get请求 Query DefaultQuery GetQueryArray

表单参数​

- 表单参数获取：
        c.PostForm("formData"), 
        file, _ := c.FormFile("file")
        DefaultPostForm
- *JSON数据绑定：
        c.ShouldBindJSON(&jsonStruct)
        c.Bind(&jsonStruct)

post请求，四种传输格式，PostForm()获取-默认解析的是x-www-form-urlencoded或者form-data格式

  1. application/x-www-form-urlencoded：通过PostForm()方法获取，默认解析的是这种格式。
  2. multipart/form-data：用于上传文件或二进制数据。
  3. application/json：用于发送JSON数据。
  4. application/xml：用于发送xml数据。
  5. text/plain：纯文本格式提交。
- 请求头信息获取：c.GetHeader("HeaderName")
- 响应状态码设置：c.Status(http.StatusOK)
- JSON响应返回：c.JSON(http.StatusOK, gin.H{"key": "value"})
- HTML模板返回：c.HTML(http.StatusOK, "index.html", gin.H{"title": "Home"})
- 重定向：c.Redirect(http.StatusFound, "/newPage")
- 字符串返回：c.String(http.StatusOK, "Hello, World!")

重定向​

r.GET("/redirect", func(c *gin.Context) {
    c.Redirect(http.StatusMovedPermanently, "/index")
})

r.GET("/goindex", func(c *gin.Context) {
	c.Request.URL.Path = "/"
	r.HandleContext(c)
})
