Go 语言中不允许将整型强制转换为布尔型.

布尔型无法参与数值运算，也无法与其他类型进行转换。

byte是uint8的别名，通常用于处理ASCII字符。 rune是int32的别名，用于处理一个UTF-8字符或Unicode码点。

byteValue := byte('A')
fmt.Println(byteValue)  // 输出: 65

runeValue := rune('你')
fmt.Printf("%#U \n", runeValue)  // 输出: U+4F60

rune 字符​

表示Unicode字符 int32类型的别名 一个Unicode字符

rune是Go语言中的一种数据类型，用于表示Unicode字符。字符用单引号（’）包裹起来

在Go中，rune实际上是一个int32类型的别名，用于表示一个Unicode码点。

因此，你可以将rune类型看作是一个整数类型，但专门用于存储Unicode字符。

在Go中，使用rune类型可以确保你能够正确处理和操作Unicode字符。

声明和初始化 单引号​

var r rune = 'a'
r1 := '世'

rune处理字符串​

[]rune(strVal) 多个字符

使用rune数组可以处理包含多字节字符的字符串，确保每个Unicode字符被正确处理

var str1 string = "你好吗123"
r2 := []rune(str1)

fmt.Printf("长度:%d  rune:%d  \n", len(str1), len(r2))
// 长度:12  rune:6

s := "Hello, 世界"
for i, r := range s {
    fmt.Printf("%d -> %c \t", i, r)
}

切片 slice​

引用类型

在Go语言中，slice是一个轻量级的数据结构，它是对数组的一个抽象，提供了更灵活的功能。以下是一些slice的相关知识点：

  • 创建slice：s := []elementType
  • 使用make()函数创建slice：s := make([]elementType, length, capacity)
  • 获取slice的长度：len(s)
  • 获取slice的容量：cap(s)
  • 切片操作：s[start:end]，包含start，不包含end
  • 追加元素到slice：s = append(s, value)
  • 复制slice：copy(destSlice, sourceSlice)
  • 删除slice中的元素：s = append(s[:index], s[index+1:]...)

map​

(一些语言的类似对象)

在Go语言中，map是一种无序的键值对集合。以下是一些map的相关知识点：

  • 使用make()函数创建map：m := make(map[keyType]valueType)
  • 添加键值对：m[key] = value
  • 获取值：value := m[key]
  • 删除键值对：delete(m, key)
  • 检查键是否存在：val, ok := m[key]
  • 迭代map：for key, value := range m 
var test = map[key的类型]val的类型

var test = map[string]int{"a": 11, "b": 22}

make(map[string]interface{})
