内置 类型&函数

内置类型​

值类型

    bool
    int(32 or 64), int8, int16, int32, int64
    uint(32 or 64), uint8(byte), uint16, uint32, uint64
    float32, float64
    string
    complex64, complex128
    array    -- 固定长度的数组

引用类型（指针类型）

    slice   -- 序列数组(最常用)
    map     -- 映射
    chan    -- 管道

内置函数​

不需要导入即可使用

    append              -- 用来追加元素到数组、slice中,返回修改后的数组、slice
    close               -- 主要用来关闭channel
    delete              -- 从map中删除key对应的value
    panic               -- 停止常规的goroutine  （panic和recover：用来做错误处理）
    recover             -- 允许程序定义goroutine的panic动作
    imag                -- 返回complex的实部   （complex、real imag：用于创建和操作复数）
    real                -- 返回complex的虚部
    make                -- 用来分配内存，返回Type本身(只能应用于slice, map, channel)
    new                 -- 用来分配内存，主要用来分配值类型，比如int、struct。返回指向Type的指针
    cap                 -- capacity是容量的意思，用于返回某个类型的最大容量（只能用于切片和 map）
    copy                -- 用于复制和连接slice，返回复制的数目
    len                 -- 来求长度，比如string、array、slice、map、channel ，返回长度
    print、println      -- 底层打印函数，在部署环境中建议使用 fmt 包

内置接口error​

//只要实现了Error()函数，返回值为String的都实现了err接口
type error interface { 
        Error()    String
}

字符串​

方法介绍
len(str)求字符串的长度
+ 或 fmt.Sprintf字符串拼接
strings.Split分割字符串，返回slice
strings.Contains检查字符串是否包含子串
strings.HasPrefix检查字符串是否以指定的前缀开头
strings.HasSuffix检查字符串是否以指定的后缀结尾
strings.Index返回子串在字符串中的索引（首次出现的位置），不存在则返回-1
strings.LastIndex返回子串在字符串中最后出现的索引，不存在则返回-1
strings.Join(a[]string, sep string)将元素类型为string的slice使用指定的分隔符拼接成一个字符串

Printf​

占位符​

占位符介绍例子
%v默认格式fmt.Printf("%v", value)
%+v类似%v，但输出结构体时会添加字段名fmt.Printf("%+v", structValue)
%#v值的Go语法表示fmt.Printf("%#v", value)
%T打印值的类型fmt.Printf("%T", value)
%t打印true或falsefmt.Printf("%t", booleanValue)
%b二进制表示fmt.Printf("%b", number)
%cUnicode码点所对应的字符fmt.Printf("%c", unicodeValue)
%d十进制表示fmt.Printf("%d", number)
%o八进制表示fmt.Printf("%o", number)
%x十六进制表示，使用a-ffmt.Printf("%x", number)
%X十六进制表示，使用A-Ffmt.Printf("%X", number)
%e科学计数法表示，使用efmt.Printf("%e", scientificValue)
%E科学计数法表示，使用Efmt.Printf("%E", scientificValue)
%f有小数点而无指数，例如123.456fmt.Printf("%f", floatValue)
%F等价于%ffmt.Printf("%F", floatValue)
%g根据实际情况选择%f或%efmt.Printf("%g", value)
%G根据实际情况选择%f或%Efmt.Printf("%G", value)
%s输出字符串fmt.Printf("%s", stringValue)
%q输出带双引号的字符串fmt.Printf("%q", stringValue)
%p输出指针地址fmt.Printf("%p", &variable)
